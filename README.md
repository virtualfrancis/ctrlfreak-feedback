# CtrlFreak Feedback

Bug reports and feature requests for [CtrlFreak](https://virtualfrancis.github.io/ctrlfreak-landing/) — a native iOS and Android homelab management app.

## Filing a Bug Report

[Open a bug report](https://github.com/virtualfrancis/ctrlfreak-feedback/issues/new?template=bug_report.md)

Please include:
- What you were doing when it happened
- What you expected vs. what actually happened
- Your device and OS version
- Which integrations you have configured (Proxmox, Portainer, etc.)

## Requesting a Feature

[Open a feature request](https://github.com/virtualfrancis/ctrlfreak-feedback/issues/new?template=feature_request.md)

## Supported Integrations

| Integration | Notes |
|---|---|
| Proxmox VE | API token auth, multi-node clusters supported |
| Portainer | API token auth, local and edge endpoints |
| Uptime Kuma | Public status page (any slug) |
| Dell iDRAC | iDRAC8 via Redfish API, Basic auth |
| SSH | Password auth, any Linux host |
